Most accounts are provisioned automatically when HR creates your employee record. This page lists what you should have and how to get access if something is missing.

## Auto-provisioned accounts

These are created before your start date and activated when you log in with your company email:

| Tool | Used for |
|------|----------|
| Company email | Primary communication |
| Slack | Team messaging |
| Project tracker | Task and sprint management |
| HR portal | Benefits, time off, payroll |
| SSO-easy | Single sign-on for all other apps |
| PasswordBuddy | Password manager |

## Request-based access

Some systems require a separate request, either because they're role-specific or have licensed seat limits:

| Tool | How to request |
|------|---------------|
| Infrastructure | File an [IT access request](/it-security/access-requests) |
| Git repository | Ask your tech lead to invite you |
| Design platform | Ask the design lead |
| Analytics tools | Request via IT ticket, manager approval required |
| Customer data systems | Manager + security approval required |

## Something missing?

File an [access request ticket](/it-security/access-requests) and include:
- The tool you need
- Why you need it
- Your manager's name for approval

Routine requests are fulfilled within one business day.
